# 🃏 Proyecto Poker en Java — Modelado de Cartas y Baraja

Este repositorio contiene el desarrollo inicial de un sistema orientado a
objetos en Java para la implementación de un juego de póker.

En esta **primera etapa**, el enfoque está puesto en la construcción del modelo de cartas y la baraja,
estableciendo una base sólida sobre la cual se desarrollará posteriormente la lógica completa del juego.

---

## Objetivo del Proyecto

El objetivo principal es construir progresivamente un sistema que permita
simular un juego de póker completo, aplicando conceptos fundamentales de
programaciónorientada a objetos como:

- Encapsulamiento.
- Herencia.
- Polimorfismo.
- Uso de enumeraciones.
- Modelado de entidades del mundo real.

Esta fase inicial se centra en representar correctamente una baraja de cartas estándar.

## Cómo usar estos ejemplos

1. **Clonar el repositorio:**
   ```bash
   git clone https://github.com/darkwhiskers/Poker_JAVA.git
   ```

2. **Compilar desde la raíz del proyecto:**
   ```bash
   javac -d . $(find src -name "*.java")
   ```

3. **Ejecutar:**
   ```bash
   java com.darkwhiskers.poker.Main
   ```
